# 自然语言开发沟通记录

## 1. 项目发起
用户提出要组织《三角洲行动》《PUBG》《CF》《CSGO》的比赛，需要一个工具检查选手电脑里是否存在违规插件、软件、硬件等，以降低开挂风险。

## 2. 需求澄清结论
经过多轮沟通，确认如下需求：
- 既需要方案，也需要实际开发工具
- 主要场景为线上远程比赛
- 检测范围包括：软件进程、外挂辅助、硬件外设、系统内核、裁判报告
- 检测流程为：赛前检查是否安全，安全后进入赛中持续监控，一直到赛后
- 风险处理采用分级处理
- 架构必须采用方案 2：用户态客户端 + 内核驱动 + 云端后台
- 允许使用内核级客户端能力

## 3. 设计阶段结论
已确认系统总体设计为：
- 选手端桌面客户端
- 用户态扫描代理
- Windows 内核驱动
- 云端规则服务与风险评分服务
- 裁判后台
- 审计与证据存储

已确认流程为：
1. 登录客户端
2. 客户端与驱动自检
3. 拉取规则并执行赛前扫描
4. 本地初判 + 云端复判
5. 生成安全评分与风险分级
6. 安全后进入赛中巡检
7. 赛后输出证据包与裁判报告
